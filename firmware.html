Ottodiy Esp32 Micropython
Web Installer

This will install Ottodiy Esp32 Micropython Runtime. To get started, connect an ESP32 Ottodiy robot to your computer and hit the "Install" button

The installer is not available because your browser does not support Web Serial. Open this page in Google Chrome or Microsoft Edge instead (but not on your iOS device).

Web installer is available only in Google Chrome and Microsoft Edge browsers.

USB Serial Driver: If you don't see your device in the list of devices, your computer might be missing the driver, get it below ⬇️ CP210x Windows ⬇️ CP210x Mac ⬇️ CP210x Linux ⬇️ CH341SER Windows ⬇️ CH341SER Mac ⬇️ CH341SER Linux Download and install, then refresh this page and try to hit the "Install" button again.

ESP Web Tools is open source.
