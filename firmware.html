Ottodiy Esp32 Micropython
Web Installer

User friendly tool to manage ESP32 Otto's in the browser:

This will install Ottodiy Esp32 Micropython Runtime. To get started, connect an ESP32 Otto to your computer and hit the Install button:

The installer is not available because your browser does not support Web Serial. Open this page in Google Chrome or Microsoft Edge instead (but not on your iOS device).

Web installer is available only in Google Chrome and Microsoft Edge browsers(but not on your iOS device).
